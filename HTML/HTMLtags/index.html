¿Qué es un elemento?

Dentro de HTML todo lo que esté entre etiquetas <ejemploEtiqueta> contenido </ejemploEtiqueta> es un elemento.

En este código se usaron las siguientes etiquetas:

  1. <!DOCTYPE html> : Que dice que este es un documento HTML5
  2. <html> Representa la rais del documento (root). El esto de los elementos desciende de este elemento.
  3. <head> Este elemento contiene la meta-informacion de la página HTML, Por lo regular el head no se muestra.
  4. <title> Este elemento está dentro del <head> e indica el titulo para la pagina HTML (Es el que se muestra en la pestaña del navegador)
  5. <body> Este elemnto, contiene todo lo visible para la pagina
  6. <h1> Define el tamño de un titulo
  7. <p> Define un parrafo
